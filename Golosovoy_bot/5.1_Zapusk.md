# 🚀 Запуск продукта (локально)

## Что значит «запуск локально»
Продукт работает на вашем компьютере. Никуда не размещён на серверах. Если закрыть Antigravity — бот перестанет работать.

## Порядок локального запуска

### Шаг 1. Установить зависимости
```
npm install
```

### Шаг 2. Настроить переменные окружения
Создать файл `.env` и заполнить:
```
OPENROUTER_API_KEY=ваш_ключ
TWILIO_SID=ваш_sid
TWILIO_AUTH_TOKEN=ваш_токен
TWILIO_PHONE=ваш_номер
ELEVENLABS_API_KEY=ваш_ключ
SUPABASE_URL=ваш_url
SUPABASE_KEY=ваш_ключ
```

### Шаг 3. Загрузить базу знаний
Положить файлы в папку `server/knowledge/`:
- products.json — каталог товаров
- faq.json — FAQ
- instructions.json — инструкции
- policies.json — политика возврата

### Шаг 4. Запустить сервер
```
npm run dev
```

### Шаг 5. Открыть админ-панель
Перейти в браузере по адресу: `http://localhost:3000`

### Шаг 6. Тестовый звонок
Позвонить на виртуальный номер и проверить работу бота.

## Ожидаемый результат
- ✅ Сервер запущен без ошибок
- ✅ Админ-панель открывается в браузере
- ✅ Бот отвечает на тестовый звонок
- ✅ Звонок появляется в журнале
