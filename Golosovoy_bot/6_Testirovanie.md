# 🧪 Тестирование

## Что тестируем

### 1. Распознавание речи
- [ ] Бот корректно распознаёт русскую речь
- [ ] Понимает разные акценты и скорости речи
- [ ] Корректно обрабатывает фоновый шум
- [ ] Не зависает при паузах в речи клиента

### 2. Качество ответов ИИ
- [ ] Бот правильно отвечает на вопросы о товарах (из базы знаний)
- [ ] Не придумывает информацию, которой нет в базе
- [ ] Корректно обрабатывает претензии (не спорит, помогает)
- [ ] Правильно проводит пошаговую диагностику
- [ ] Помнит контекст разговора (не переспрашивает)

### 3. Голос бота (TTS)
- [ ] Голос звучит естественно
- [ ] Правильное произношение названий товаров
- [ ] Задержка ответа не более 3 секунд
- [ ] Нет обрывов и артефактов звука

### 4. Переключение на оператора
- [ ] Бот переключает при прямой просьбе клиента
- [ ] Бот переключает при 3+ неудачных попытках помочь
- [ ] Оператор получает контекст разговора
- [ ] Клиент не теряется при переключении

### 5. Логирование
- [ ] Все звонки записываются
- [ ] Транскрибация сохраняется полностью
- [ ] В журнале отображается корректная информация
- [ ] Фильтры в админ-панели работают

### 6. Админ-панель
- [ ] Все страницы открываются без ошибок
- [ ] Статистика отображается корректно
- [ ] Можно прослушать запись и прочитать транскрибацию

## Как тестируем
1. Позвонить на номер бота и задать типовые вопросы
2. Проверить ответы на соответствие базе знаний
3. Попробовать сложные/необычные вопросы
4. Пожаловаться на «сломанный товар» — проверить сценарий претензии
5. Попросить переключить на оператора
6. Проверить, что всё залогировалось в админке
