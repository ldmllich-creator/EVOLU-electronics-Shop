# 🐛 Дебаггинг (исправление ошибок)

## Типичные проблемы и как их решать

### 🔴 Бот не отвечает на звонок
**Возможные причины:**
- Нет подключения к Twilio → проверить API-ключи
- Сервер не запущен → запустить `npm run dev`
- Вебхуки некорректные → проверить URL в настройках Twilio
**Как исправить:** Написать в Antigravity: «Бот не отвечает на звонки, проверь подключение к телефонии и вебхуки»

### 🔴 Бот не понимает речь
**Возможные причины:**
- Проблема с Whisper API → проверить ключ
- Плохое качество аудио → проверить настройки кодека
- Язык распознавания → убедиться, что стоит русский
**Как исправить:** «Проверь модуль STT, бот не распознаёт речь»

### 🔴 Бот даёт неправильные ответы
**Возможные причины:**
- Некорректный промпт → переписать системный промпт
- Устаревшая база знаний → обновить данные товаров
- Нет нужной информации в базе → дополнить
**Как исправить:** «Обнови промпт для ИИ, бот даёт неточные ответы о товарах»

### 🔴 Большая задержка ответа (> 5 сек)
**Возможные причины:**
- Медленная модель → переключить на более быструю (MiniMax)
- Медленный TTS → сменить провайдер
- Перегруз сервера → увеличить ресурсы
**Как исправить:** «Оптимизируй скорость ответа бота, задержка больше 5 секунд»

### 🔴 Не сохраняются логи
**Возможные причины:**
- Нет подключения к Supabase → проверить URL и ключ
- Таблицы не созданы → создать схему БД
- Ошибки записи → проверить формат данных
**Как исправить:** «Проверь подключение к Supabase, звонки не логируются»

### 🔴 Админ-панель не открывается
**Возможные причины:**
- Ошибка в HTML/JS → проверить консоль браузера (F12)
- Сервер не отдаёт статику → проверить настройку Express
- CORS-ошибка → добавить CORS-заголовки
**Как исправить:** «Админ-панель не загружается, проверь настройки сервера и CORS»

## Общие правила дебаггинга

1. **Описать проблему** максимально точно в чате Antigravity
2. **Скопировать ошибку** из терминала — вставить в чат
3. **Не паниковать** — нейронка разберётся
4. **Откатиться назад** (Reject All), если стало хуже
5. **Один баг = один запрос** — не пытаться починить всё разом
