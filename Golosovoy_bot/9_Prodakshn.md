# 🏁 Продакшн (Релиз)

## Что значит «продакшн»
Продукт полностью готов, протестирован, задеплоен и работает в боевом режиме с реальными клиентами.

## Чеклист перед релизом

### Функциональность
- [ ] Бот корректно принимает все входящие звонки
- [ ] Распознаёт речь с точностью > 90%
- [ ] Отвечает на основе загруженной базы знаний
- [ ] Корректно обрабатывает претензии
- [ ] Переключает на оператора при необходимости
- [ ] Все звонки логируются

### Стабильность
- [ ] Работает без падений > 48 часов подряд
- [ ] Выдерживает 5+ одновременных звонков
- [ ] Автоматически перезапускается при сбое (PM2)
- [ ] Задержка ответа < 3 сек стабильно

### Безопасность
- [ ] API-ключи не видны в коде (только .env)
- [ ] Админ-панель защищена паролем
- [ ] HTTPS включён
- [ ] Данные клиентов защищены

### Контент
- [ ] База знаний заполнена реальными данными
- [ ] Промпт ИИ проверен и финализирован
- [ ] Тексты приветствия/прощания утверждены
- [ ] Голос бота выбран и настроен

### Мониторинг
- [ ] Настроены уведомления о падении сервера
- [ ] Ежедневный отчёт по звонкам
- [ ] Алерты при аномалиях (резкий рост звонков, ошибки)

## После релиза
1. **Первая неделя** — внимательно следим за всеми звонками, читаем транскрибации
2. **Собираем обратную связь** — что клиенты спрашивают, на что бот не может ответить
3. **Дополняем базу знаний** — добавляем новые вопросы и ответы
4. **Улучшаем промпт** — на основе реальных диалогов
5. **Масштабируем** — увеличиваем количество одновременных линий
