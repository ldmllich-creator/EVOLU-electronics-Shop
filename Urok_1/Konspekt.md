## 🧠 Что такое вайбкодинг?

Создание цифровых продуктов (сайтов, ботов, приложений) **без глубоких знаний программирования** — с помощью ИИ. Ты ставишь задачу, ИИ пишет код. Единственный критерий: **работает или не работает**.

---

## 🔄 Цикл разработки (9+1 шагов)

```
0. Исследование  →  Анализ рынка и потребностей
1. Идея           →  Что за продукт и зачем
2. ТЗ             →  ИИ расписывает функциональность
3. Архитектура    →  Структура файлов и папок
4. Дизайн         →  Как будет выглядеть (UI/UX)
5. Задачи         →  Чеклист для ИИ
   5.1 Запуск     →  Работает на компьютере
6. Тестирование   →  Ищем баги
7. Деплой         →  На сервер
8. Дебаггинг      →  Исправляем ошибки
9. Продакшн       →  Релиз!
```

> **Формула успеха:**  
> 40–60% — качество документации (шаги 1–5)  
> 20% — сама разработка  
> 40% — доработка и исправление ошибок

## 💻 Antigravity — наш инструмент

| Элемент | Где | Что делает |
|---------|-----|-----------|
| **Чат с ИИ** | Справа | Пишем запросы и получаем код |
| **Файлы** | Слева | Дерево файлов проекта |
| **Просмотр** | По центру | Предпросмотр результата |
| **Accept All** | Кнопка | Принять изменения ИИ (зелёное) |
| **Reject All** | Кнопка | Откатить (отклонить красное) |
| **История** | Иконка часов | Прошлые переписки |

### Правила работы:
- 📁 Хранить файлы в порядке — понятные имена, структура папок
- 📎 Перетаскивать файлы/папки в чат — ИИ увидит содержимое
- 🛡️ Не снимать ограничители — подтверждать действия вручную
- 📝 Писать в промпте **«Не пиши код»** — пока формируем архитектуру
- ⚙️ Глобальный промпт — правила для ИИ на уровне всей программы

## 🔒 10 правил безопасности

1. ИИ **не автономен** — ты решаешь, куда ему можно
2. **Не удалит** файлы с компьютера
3. Работа идёт **в отдельных папках**
4. Серверные изменения — **отдельно**
5. **Глобальный промпт** = правила поведения ИИ
6. **Проверяй** предложения ИИ перед принятием
7. Попроси ИИ **расписывать действия подробно**
8. Используй **отдельный компьютер** / виртуалку для рискованных задач
9. **Делай бэкапы** (клонирование, GitHub)
10. Настрой **скиллы безопасности**
